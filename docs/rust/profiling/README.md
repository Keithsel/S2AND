# Profiling snapshots (Rust)

This folder holds **dated** profiling refreshes and comparisons. These are point-in-time
evidence and are not “active baselines” unless promoted in `docs/rust/baselines.md`.

Guidelines:

- Name files by date: `YYYY-MM-DD.md`.
- Include: environment, commands run, produced artifacts, and a short interpretation.
- Prefer linking to `scratch/*.json` artifacts rather than pasting large logs inline.
