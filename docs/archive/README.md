# Archive (forensics only)

This folder is for historical notes and one-off investigation writeups that are no longer part of
the active execution plan.

Active sources of truth:
- Work bundles / where to start: `docs/work_plan.md`
- Rust baselines + promotion rules: `docs/rust/baselines.md`
- Rust roadmap / worklist: `docs/rust/roadmap.md`
- Dated profiling snapshots: `docs/rust/profiling/`

Guidelines:
- Prefer linking to `scratch/` artifacts rather than pasting long logs inline.
- Keep archive docs immutable once superseded; add a new dated note instead of rewriting history.
